Ricoline

Función:

Ecommers de la empresa de dulces Ricolino, esta página web fue diseñada para la fácil distribución y venta de los dulces; tiene un inicio de sesión, un apartado de promociones y categoría, la opción de guardar en el carrito y pagar por el método de mercado libre


Herramientas:

React, NodeJs, Express, AlwaysData

Control de medicamentos

Función:

Página web que ayuda a tener un control de días y horas de los medicamentos añadidos por el usuario. Tiene las funciones de añadir medicamentos y este al momento de la primera toma va midiendo la siguiente toma y cuando llegue a la dosis recetada desaparece el medicamento


Herramientas:

React, NodeJs, Express, AlwaysData

Hora de pasear

Función:

Página web que le brinda al usuario una mejor experiencia a la hora de pasear por la ciudad de Cancún. Para el inicio de sesión se utilizó un api con el que puedes acceder directamente desde tu cuenta de Google.

Para el contenido de la página se utilizó tres apis una de clima, uno de lugares (Google Maps) y por ultimo uno de noticias; todas estas apis brindan información de la ciudad de Cancún


Herramientas:

React, NodeJs, Express, AlwaysData, Firebase

Taller mecanico

Función:

Dashboard que ayuda a un taller de mecánicos a tener más control de sus trabajos por realizar, brindándoles un apartado para que registren sus trabajos, un control de sus horas trabajadas y precios de materiales utilizados; esto para que le genere el total de su trabajo dependiendo del tipo de trabajo realizado.


Herramientas:

React, NodeJs, Express, AlwaysData
